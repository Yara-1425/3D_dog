# Robo-Dog 3D CAD Design

A simple 3D model of a robotic dog designed and assembled from scratch using **Onshape**. 

## Features
* **Sensor Ready:** The head includes dual circular slots for robotic sensors.
* **2-Part Legs:** 4 legs, each split into two separate components to simulate realistic joints.

##  Components
* **1 x Main Body:** The core chassis with rounded corners.
* **1 x Head Unit:** Features pre-designed sensor cutouts.
* **8 x Leg Segments:** Two connected parts per leg (4 legs total).
